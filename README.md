# GNS-BOT
## Meet Sofía
---
**GNS-BOT** is a customer support **AI powered widget** for an internet provider's (GNS) CRM. Ready for deployment in the company's server, it embeds into the CRM with a single HTML tag and lets customers troubleshoot connectivity issues and manage support tickets without having to call a human agent. It acts as an intermediary between the user and GNS's API to create, manage and update tickets to speed-up network problems fixing processes.

**"Sofía"** will be GNS's clients personal agent, powered by Gemini 3.1 flash-lite (or any other Gemini model) to do this specific tasks: 

 + Opening and saving sessions in cache.
 + Authenticate users to begin their support process.
 + Diagnose network problems by walking the cusutomer through step-by-step troubleshooting before escalating to a technician.
 + Open support tickets automatically, writing a detailed, technnical desription of the reported issue and the attempted fixing protocols.
 + Look up existing tickets so the customer can check status of previous reports and complement their tickets with comments.
 + Inform about pending balance if the service has been suspended to an outstanding payment.
